<h1>HowToSecurity: An interactive educational security platform.</h1>
<body>
  HowToSecurity is an interactive web-based learning platform for software developers to learn software security concepts such as password storage, SQLi, etc. in an engaging and clear way. 
  It aims to bridge the gap between software developers and cybersecurity professionals, allowing developers to teach themselves skills which are important to anyone in tech,
  not just cybersecurity professionals.
</body>
